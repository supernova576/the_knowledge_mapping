Learning Question Format Help

Add or edit custom questions by changing the two JSON boxes and pressing Save.

Question types

  • FREETEXT: open answer field, shown in review, not counted in the final score.
  • SINGLE_CHOICE: exactly one correct answer, requires an options list.
  • MULTIPLE_CHOICE: one or more correct answers, requires an options list.

Rules

  • Every question needs a unique id.
  • Every question needs a matching answer entry in the answers JSON.
  • For choice questions, answer values must exactly match the option text.
  • For free-text questions, omit options.

Example

{
  "questions": [
    {
      "id": "q001",
      "type": "SINGLE_CHOICE",
      "text": "Which status allows AI feedback generation?",
      "options": ["Draft", "Under Construction", "Done"]
    },
    {
      "id": "q002",
      "type": "MULTIPLE_CHOICE",
      "text": "Which fields belong to a learning question?",
      "options": ["id", "type", "text", "options"]
    },
    {
      "id": "q003",
      "type": "FREETEXT",
      "text": "Summarize when a free-text question should be used."
    }
  ]
}
{
  "answers": [
    {
      "question_id": "q001",
      "correct_answers": ["Done"]
    },
    {
      "question_id": "q002",
      "correct_answers": ["id", "type", "text", "options"]
    },
    {
      "question_id": "q003",
      "correct_answers": ["Use free-text for explanation, reasoning, or summary answers."]
    }
  ]
}
