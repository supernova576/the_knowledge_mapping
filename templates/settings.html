Compliance Check

Configure each compliance rule separately. Every rule has its own Enable toggle and related parameters.

1) Structure Rule

All listed strings must exist in the document.


2) Created Date Rule

Requires a valid "Erstellt" date in the document metadata.


3) Beschreibung Rule

The description section must not exceed the configured sentence limit.


4) External Links Rule

Requires at least this number of external links.


5) Tags Rule

Requires at least this number of tags in "Page Tags".


6) Video Links Rule

If document length is above this threshold, at least one video link is required.


7) AI Feedback Score Rule

Uses the latest AI feedback score of the note. If the score is below the threshold, the note is marked incompliant.
